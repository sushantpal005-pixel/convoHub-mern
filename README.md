
# 💬 ConvoHub – Real-Time Chat Application

ConvoHub is a full-stack real-time chat application that allows users to communicate instantly with a modern UI, real-time updates, and persistent state management.
* Live link : https://convohub-ju3o.onrender.com/
---

## 🚀 Features

* 🔐 User Authentication (Login / Signup)
* 💬 Real-time Messaging using Socket.io
* 🟢 Online Users Indicator
* ⚡ Fast and Responsive UI
* 🧠 Global State Management with Redux Toolkit
* 💾 State Persistence using Redux Persist
* 🎨 Styled with Tailwind CSS & DaisyUI
* 🔔 Toast Notifications for better UX

---

## 🛠️ Tech Stack

### Frontend

* React.js + vite
* Redux Toolkit
* Redux Persist
* Tailwind CSS + DaisyUI
* Axios
* Socket.io-client

### Backend

* Node.js
* Express.js
* Socket.io
* MongoDB Atlas
